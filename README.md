# Courier Service Backend

## Description

[Nest](https://github.com/nestjs/nest) framework-based courier delivery service backend application.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Scripts

The project has multiple scripts to manage admin users and count database data. See `package.json` for the details.
